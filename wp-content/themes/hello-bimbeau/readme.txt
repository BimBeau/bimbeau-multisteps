A lightweight and minimalist WordPress theme for Elementor page builder.

== Description ==

Hello BimBeau is a lightweight and minimalist WordPress theme that was built specifically to work seamlessly with the Elementor page builder plugin. The theme is free, open-source, and designed for users who want a flexible, easy-to-use, and customizable website.
